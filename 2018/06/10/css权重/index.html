css 中有不同的选择器，不同的选择器拥有不同的优先级

!important Infinity //优先级最高

行内样式 1000

id 100

class |属性|伪类 10

标签|伪元素 1

通配符 0

作用在同一个元素上的选择器，优先级高的生效。 如果优先级相同则靠后的生效

注意10个标签选择器优先级没有1个class选择器高，选择器之间并不是简单的十进制，有资料说进制为256。
