树莓派实现远程视频监控

2018.06.09 zmkm  热度 ℃
  • 准备
    树莓派，usb摄像头
  • 使用montion
    • 安装montion
      sudo apt-get install motion
    • 修改motion配置
      sudo nano /etc/motion/motion.conf
      将on改为off
      stream_localhost off
      这样就可以在局域网的其他设备上访问motion服务
      其他配置还可以修改摄像画面的分辨率等
    • 启动motion
      sudo motion

在局域网里打开浏览器，在地址栏输入树莓派的ip地址端口号默认为8081,查看监控画面
