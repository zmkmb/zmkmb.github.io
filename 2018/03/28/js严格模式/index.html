js严格模式

如何使用

在脚本前加”use strict”;

有哪些变化

1.变量必须显式声明

2.this默认是undefined

3.严格模式要求函数的参数名唯一

4.不可以用with

5.禁止八进制数字语法

6.不再支持 arguments.callee

7.参数的值不会随 arguments 对象的值的改变而变化

。。。

注意事项

如果合并脚本，严格模式会导致，一些没用采用严格模式的脚本出错，所以使用时最好用一个函数把代码包裹器来。

https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Strict_mode
