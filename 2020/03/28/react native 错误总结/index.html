错误Invalid escape sequence at line 1 column 29 path $[0].name

项目在android studio下运行正常，在命令行执行react-native run-android报错

  • 在android studio 在 Build菜单下执行clean project
  • 或者在android目录，执行 gradlew clean

Attempt to invoke virtual method ‘double java.lang.Double.doubleValue()’ on a null object reference
