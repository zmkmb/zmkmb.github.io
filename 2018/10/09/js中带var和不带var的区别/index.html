js中带var和不带var的区别

2018.10.09 zmkm 前端  热度 ℃

js中使用var来声明一个变量。

并且在全局作用域下变量是window的一个属性

如果变量没有使用var声明，相当于在window上添加了一个属性。

两者的区别是var声明的变量会有变量提升。

严格模式下一个变量没有声明就被赋值，js会执行错误。

var a = 1,b=2;相当于var a = 1;var b = 2;

var a = b = 1;相当于var a = 1;b = 1;
