flex布局

flex是CSS3中新添加的属性。比起之前的浮动布局更具有灵活性

在不考虑IE,及低版本安卓系统的兼容下，可以放心使用

  • flex容器属性
    • flex-flow
    • flex-direction
    • flex-wrap
    • justify-content
    • align-items
    • align-content
  • flex元素属性
    • order
    • flex-grow
    • flex-shrink
    • flex-basis
    • flex
    • align-self

首先每一个弹性容器都有两根轴，主轴和交叉轴

  • flex-direction

    flex-direction用来设定主轴的方向,容器内的子元素会根据主轴方向来排列

    • row(默认):水平
    • column：垂直
    • row-reverse
    • column-reverse

  • flex-wrap

    flex-wrap用来设定，容器中的子元素在主轴方向排不下的情况向如何显示

    • nowrap(默认)：不换行
    • wrap：折行
    • wrap-reverse：反向折行

  • flex-flow

    复合属性相当于flex-direction + flex-wrap

  • justify-content

    主轴方向的对齐方式

  • align-items

    交叉轴方向的对齐方式

  • flex-order

    设置元素之间的排序，数值越小越靠前

  • flex-grow

    当flex-wrap为nowrap时，子元素在主轴方向没有排满时，可以用flex-grow来控制子元素的显示

    默认是0，不放大

  • flex-shrink

    当flex-wrap为nowrap时，子元素在主轴方向排满超出时，可以用flex-shrink来控制子元素的显示

    默认是1，按比例缩放

  • flex-basis

    flex-basis设置元素在主轴上的初始尺寸，也就是flex-grow和flex-shrink生效前的尺寸

    • 如果为0，根据内容撑开
    • 如果非0，优先级大于width
    • 如果为auto，优先取width，没有则根据内容决定

  • flex

    flex-grow, flex-shrink, flex-basis 的复合属性

    flex:1 = flex:1 1 0

    flex:auto = flex:1 1 auto

    flex:none = flex:0 0 auto
