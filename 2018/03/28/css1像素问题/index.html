css 1像素问题

2018.03.28 zmkm  热度 ℃

为什么会出现1像素问题？

高分辨率的手机，用多个物理像素表示css中的1px。

我们打开谷歌浏览器，开启手机模拟模式，选择iphone6,可以看到iphone6的分辨率是375 667，但是iphone6屏幕的分辨率是750 1334。

我们称 375 667 为逻辑像素,750 1334 为物理像素

也就是说iphone6上，4个物理像素显示1个逻辑像素。
这样就导致我们用css声明的1px边框看起来显的粗一些，这并不是我们想看到。

如何解决？

  • ios8以上的版本支持0.5px

缺点：存在兼容性问题

  • 设置 border-image

  • 伪元素缩放

  • 页面整体缩放
